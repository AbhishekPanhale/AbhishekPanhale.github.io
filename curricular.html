    Just Chefin!

    Project Description:
    Just Chefin! is a stealth game in which the player takes the role of a chef that infiltrates various restaurants in order to steal their signature recipe, all while fulfilling the orders that are provided to them by the wait staff.
    Game Engine: Unity
    Implementation using: C# in Visual Studio
    Learning:
    This was my capstone project where I was a part of a team of 3 members consisting of a lead generalist, a designer and a programmer (myself). This experience was both fun and frustrating as I learnt a lot about how important it is for a game to have a solid concept that can be convincingly pitched and where different aspects of design tie together well. I was responsible for implementing the player movement, environmental interactions of the player and the enemy behavior. I also coded some gameplay aspects like fog of war, fading walls, and animation as well as particle effects for fire alarms and pickups respectively. This project enabled me to apply the State Pattern design pattern that I had tried in my other projects, for the enemy behaviour.

Enemy AI - State Pattern

 

Enemy AI Simulation!

  • ×

    Enemy AI - State Pattern

    Project Description:
    The simulation consists of an enemy (capsule) consisting of 4 states, namely Patrol, Idle, Approach and Attack. The enemy patrols through a set of points (idling at each point for some time), looking for the player. Once the player is detected, its state changes to approach. If the player is close enough to the enemy, its state changes to attacking, and the player loses health.
    Game Engine: Unity
    Implementation using: C# in Visual Studio
    Learning:
    The behaviour was achieved using the State Pattern which is an Object-oriented approach for using Finite State Machines. Each state is a class of its own, and inherits from a parent State class. The enemy itself keeps track of what its current state is.

    Project Description:
    ECS Space Simulation consists of a rotating Sun and moving asteroids, implemented using the Entity-Component-System of Unity.
