A 3D shooter game where the player moves around in a restricted space and shoots down boxes with different attributes to score points before the timer runs out.
Apart from the usual C# scripting, I got to explore the Animator component of the Unity Engine.

Roller Madness

A 3D game where, as a roller ball, you collect required number of coins, while avoiding contact with enemy boxes trying to tag you, in order to clear the level.
This is a project where I explored implementing some movement mechanics, interactive UI and level design using Unity Engine.

Pogo Santa

A 2D platformer prototype consisting of jumping on platforms that disappear once stepped upon, and collecting pickups to increase the score.
My very first attempt at game development using Unity. Tried to incorporate aspects of different tutorials that I followed.

Solar System Simulation

A simulation consisting of a rotating Sun and few planets rotating around it.
A project made using Unity where I learnt using quarternions to rotate objects. Also, I got the opportunity to implement mutiple interactive cameras. i.e. clicking on a planet shifts the main camera focus to that planet.

Bulls & Cows

A simple text-based game to guess a 6-letter isogram within limited chances. The number of bulls is the number of letters at correct positions whereas the number of cows is the number of correct letters at incorrect positions
The project made me understand some fundamental concepts in OOP using C++.
Note: Please use XCode on MacOS to run the code.
