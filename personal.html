    Action Adventure Project

    Project Description:
    A third-person action adventure prototype where you make your way by fighting enemies using weapons, and avoiding explosives. Use potions to heal yourself, and collect coins for score.
    Game Engine: Unreal Engine 4
    Implementation using: C++ in Visual Studio
    Learning:
    This is a part of the Udemy course 'Unreal Engine C++ The Ultimate Game Developer Course', that I was following. This project has personally been a major milestone for me as it has got me started with Unreal Engine, for real. Although this isn't a complete game, I was able to learn how different game mechanics are implemented using Unreal C++ and Blueprints. Below is a list of things that I learnt essentially:

    • Character Movement, Combat, and integrating corresponding animations
    • Weapons, Spawn Volumes, Pickups, Explosives, Floating Platforms
    • Enemy AI, Combat, and integrating corresponding animations
    • Level Change and Basic Save/Load game
    • Player HUD
    • Pause Menu UI

    Bulls & Cows

    Project Description:
    A simple text-based game to guess a 6-letter isogram within limited number of chances. The number of bulls is the number of letters at correct positions whereas the number of cows is the number of correct letters at incorrect positions.
    Game Engine: Unreal Engine 4
    Implementation using: C++ with XCode
    Learning:
    This project was a part of a udemy course 'Unreal Engine C++ Developer: Learn C++ and Make Video Games', that I was following. Though it was an Unreal Engine project, it was completely a console-based. It made me understand some fundamental concepts in Object Oriented Programming using C++.
